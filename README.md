# A Flutter "Todo" App

This project is part of a talk introducing Flutter to new developers.

### Requirements
- Listing items
- Adding items
- Removing items
- Editing items
- Data persistence

### Functionality
- Animated list
- Shared Preferences (Data Persistence)
- Unit tests
- Widget tests
- Integration tests (Automation)
